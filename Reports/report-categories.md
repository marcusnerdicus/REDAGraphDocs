## Report Categories

Report Categories provide a reference for what data may be included in a Report Type and the associated Report's output. The Report Category also determines which Report Criteria are available for data filtering. For example, most Report Types falling under the All Areas Category do not provide Area-level filtering in Report Criteria.

Report Categories include:

- All Areas: Reports using all the MLS Areas in a Tenant's data set in the report calculations
- Dashboard: Reports based on a corresponding graph in the Dashboard
- Neighborhood: Reports using MLS Area or Area Groups in the report output. Not all report types support Area Groups
- Agency: Reports with a focus on overall Market conditions or agency-level comparisons from the MLS data
- Rental: Reports for Listings where the property is available for Rent or Lease
