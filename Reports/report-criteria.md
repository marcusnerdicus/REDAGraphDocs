## Report Criteria

Report Criteria is used by some but not all Report Types to filter the data used in a selected Report's output. The Report Type determines which filtering criteria is available for a Report. Some Reports may only allow a Price Range selection, while other Reports may allow filtering by the number of Bedrooms, Bathrooms, and property Square Feet. 

The example below shows all the Report Criteria filters available. The selected Report Type determines which of the filters are displayed and available for use. If a User modifies the Report Criteria then saves the criteria, the Report is re-rendered using the new criteria. When a user opens a Report from either the mobile app or the web app, the Report Criteria last saved will be used for the Report output.

The **Report Category** will determince which reports are available for selection.

The **Report Type** also determince which reports are available for selection, after the Category is selected.

The **Report Name** is required and defaults to the Category and Type but can be edited by the User. A good descriptinve name is useful when reviewing a list of reports for use later.

The **Report Time Period** may or may not be present, depending on the Report Type selected.

The **Report Price Ranges** may or may not be present, depending on the Report Type selected.

The **Report Advanced Filters** include the number of Bathrooms, Bedrooms, and Square Footage. The Advanced Filters are not available on all reports. 

The **Area Group Option** may or may not be present, depending on the Report Type selected.

The **Search** field in the Report Criteria dialog is used to search for MLS Areas in the Tenant's Area (Neighborhood) list, if used by the report.

The **Area IDs Selected** text displays the Areas a user has selected to be included in the Report output. 

The **Report Time period** allows the user to specify how many days will be included in the Report output.

The **Filter Begin Date** and **Filter End Date** fields allow a user to specify dates if they want something other than the Report Time period values.
